# 介绍

# 常用命令

```bash
# 创建python虚拟环境
python -m virtualenv venv

# 在当前终端会话中启用python虚拟环境（在cmd命令提示符下）
venv\Scripts\activate

# 从依赖要求安装所需依赖
pip install -r requirements.txt

# 把当前已安装的pip包写入到依赖要求
pip freeze > requirements.txt
```
